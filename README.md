# [mgnbar.info](https://mgnbar.info)

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.21296960.svg)](https://doi.org/10.5281/zenodo.21296960)

This is the repository for a website on the moduli space of stable marked
curves $\overline{\mathrm{M}}_{g,n}$, and its beautiful geometric properties.

The website is an interactive table indexed by the genus $g$ and the number of
marked points $n$. A selector switches between several layers:

* the **Kodaira dimension** of $\overline{\mathrm{M}}_{g,n}$, refined on the
  negative-Kodaira-dimension cells by the **rationality hierarchy** (rational,
  unirational, uniruled);
* whether the **Chow ring** is generated by tautological classes;
* whether the **point counts** over finite fields are polynomial.

Each layer has its own URL, and clicking a cell shows the value together with
its references, linked to MathSciNet or the arXiv. See the
[about page](https://mgnbar.info/about/) for how to cite the website.

It is a static website, generated with [Hugo](https://gohugo.io) and served
from GitHub Pages.

Please use [GitHub issues](https://github.com/pbelmans/mgnbar/issues)
to suggest information you want to see displayed, or get in touch with
one of us through the contact information below.

## How to cite

The website is archived on Zenodo. To cite the version-independent latest
release, use the concept DOI
[10.5281/zenodo.21296960](https://doi.org/10.5281/zenodo.21296960).

```bibtex
@online{mgnbar,
  author = {Barros, Ignacio and Belmans, Pieter},
  title  = {Mgnbar.info: the geometry of the moduli space of curves},
  url    = {https://mgnbar.info},
  doi    = {10.5281/zenodo.21296960},
  year   = {2026},
}
```
