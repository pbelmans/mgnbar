## About

**Mgnbar.info** is an interactive overview of what is known about the geometry
of the moduli space of stable *n*-pointed genus *g* curves and of its interior,
the moduli space of smooth pointed curves. Choose a layer above the table to
switch between invariants: the Kodaira dimension, whether the Chow ring is
generated by tautological classes, and whether point counts over finite fields
are polynomial. Click a cell to read off the value together with its references.

The website started as a static table of the Kodaira dimension made by
[Ignacio Barros](https://sites.google.com/view/ibarros/home), and was extended
into this interactive resource by [Pieter Belmans](https://pbelmans.ncag.info).
The source code and the underlying data live on
[GitHub](https://github.com/pbelmans/mgnbar), where suggestions and corrections
are very welcome.

## How to cite

The website is archived on Zenodo. To cite the version-independent latest
release, use the concept DOI
[10.5281/zenodo.21296960](https://doi.org/10.5281/zenodo.21296960).

```
@online{mgnbar,
  author = {Barros, Ignacio and Belmans, Pieter},
  title  = {Mgnbar.info: the geometry of the moduli space of curves},
  url    = {https://mgnbar.info},
  doi    = {10.5281/zenodo.21296960},
  year   = {2026},
}
```
